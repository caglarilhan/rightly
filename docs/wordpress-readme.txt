# GDPR Hub Lite for WooCommerce

WooCommerce için otomatik GDPR DSAR yönetimi – erişim, silme, veri ihracı.

## Description

GDPR Hub Lite, WooCommerce mağazanızda GDPR DSAR (Data Subject Access Request) taleplerini otomatik olarak işler. Veri erişim, silme ve ihracı taleplerini 30 gün içinde GDPR uyumlu şekilde tamamlar.

### Features

- **Otomatik DSAR Portal**: Müşterilerin veri taleplerini kolayca yapabilmesi
- **Veri Erişimi**: Müşteri verilerini JSON/CSV formatında export
- **Veri Silme**: GDPR uyumlu veri silme işlemleri
- **Denetçi Raporları**: PDF formatında hazır raporlar
- **Audit Trail**: Tüm işlemlerin kayıt altına alınması

### Installation

1. Upload plugin to `/wp-content/plugins/gdpr-hub-lite/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > GDPR Hub Lite to configure

### Usage

#### Shortcode
Add DSAR portal to any page:
```
[gdpr_hub_portal]
```

#### Admin Menu
- WooCommerce > GDPR Hub Lite > Settings
- WooCommerce > GDPR Hub Lite > Requests
- WooCommerce > GDPR Hub Lite > Reports

### Requirements

- **WordPress**: 6.0 or higher
- **PHP**: 7.4 or higher
- **WooCommerce**: 6.0 or higher
- **Admin Capability**: shop_manager
